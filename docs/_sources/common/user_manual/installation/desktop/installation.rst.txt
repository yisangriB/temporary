************
Installation
************

All SimCenter applications can be downloaded from the |ResearchTools| page. In order to install and run the application, it is necessary to first download and install a **Python3** interpreter. This chapter walks you through the installation process on Windows and macOS operating systems from downloading the application to testing that it has been installed correctly.
